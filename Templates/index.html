Movies Recommendation System
Machine Learning Project

A movie-recommedation-system is used in some streaming platforms like Netflix, Amazon, Hotstar, Youtube etc. It recommends movies according to the users activity or searches(by Genres, Keywords, Tags).

STEP 1

Get the data. Then open it using Jupyter Notebook or VS Code or any other IDE.

STEP 2

Data Cleaning : If there is any duplicate values or not . And deleting the unwanted data.

STEP 3

Data Processing : Processing the data to make Tags using the columns for quicker recommendation.

STEP 4

Vectorization & Similarity : The data will be converted to vectors to get the closest vectors/ similarity between the vectors according to the searches.

FINISH

Make a Recommend function.Then find a way to deploy it, if the system works.
